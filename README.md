# Vrinda Store Data Analysis (Interactive Dashboard using MS Excel)

## Project Objective
The objective of this project is to analyze Vrinda Store's sales performance for the year 2022 and identify customer purchasing behavior, sales trends, and growth opportunities. The insights generated from this analysis can help the business improve sales and make data-driven decisions for 2023.

---

## Dataset
- Vrinda Store Sales Data (2022)

---

## Business Questions (KPIs)

1. Compare Sales and Orders using a single chart.
2. Which month recorded the highest Sales and Orders?
3. Who purchased more products: Men or Women?
4. What were the different Order Status categories in 2022?
5. Which Top 10 States contributed the most to Sales?
6. What is the relationship between Age Group and Gender based on Orders?
7. Which Sales Channel contributed the most revenue?
8. Which Product Category generated the highest sales?

---

## Data Cleaning & Preparation

- Checked and handled missing values.
- Removed inconsistencies and data anomalies.
- Standardized data types and formats.
- Verified data accuracy and quality.
- Created Pivot Tables for KPI analysis.
- Built an interactive dashboard using charts and slicers.
